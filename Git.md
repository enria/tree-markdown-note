#### 添加到提交分组

```shell
git add -A
```

#### 提交

```shell
git commit -m "commit message"
```

#### 推送到远程

```shell
git push origin master
```

#### 拉取到本地

```shell
git pull 
```

#### 中文乱码

```shell
git config --global core.quotepath false
```

#### 记住密码

```shell
git config --global credential.helper store

1
```
