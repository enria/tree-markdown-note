#### 通过字段长度排序

```http
GET base_enterprise_info/base_enterprise_info/_search
{
  "sort": {
    "_script": {
      "type": "number",
      "script": {
        "lang": "painless",
        "inline": "doc['company_name.keyword'].size()==0?0:doc['company_name.keyword'].get(0).length()"
      },
      "order": "asc"
    }
  }
}

```
